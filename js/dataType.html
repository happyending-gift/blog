数据类型分类

基本数据类型

number 数字型

介绍

Javascript是弱数据类型，赋值后才能判断

  • 整数
  • 小数
  • 负数

string 字符串型

字符串含义

字符串使用英文状态下的" "或' ' 单引号和双引号可以相互嵌套，但注意不能自己嵌套自己

字符串拼接
document.write('忍者'+'神龟')

let comic = '火影忍者'
let author = '岸本齐史'
document.write(comic + author)
模板字符串
let comic = '火影忍者'
let author = '岸本齐史'

document.write(${comic}的作者是${author})

Boolean 布尔型

let isDarkMode = true;

undefined 未定义型

只声明变量，未赋值

null 空

声明并赋值，但为空

引用数据类型

object 对象

function 函数

array 数组

检测数据类型

let num = 12;
console.log(typeof num);

数据类型转换

隐式转换

let num = 12;
console.log(typeof num + 12);  //输出number12

+两边只要有一个字符串，都会把另外一个转成字符串 除了+以外的算术运算符，比如- * /等都会把数据转成数字类型
